# Security Control Mapping

| Threat | Defensive Control | Status |
|-------|------------------|--------|
| Brute Force | Fail2Ban | Partially validated |
| Password Guessing | Account lockout | Conceptual |
| Credential Abuse | Strong password policy | Recommended |
| Repeated SSH Attempts | Log monitoring in Splunk | Validated |
| Malicious Source Detection | Source IP correlation | Validated |
