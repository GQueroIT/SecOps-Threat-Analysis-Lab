## What Worked

- Brute-force activity was successfully simulated
- Splunk correlated failed authentication events
- Source IP activity was identified
- Preventive control was deployed and validated

## What Was Challenging

- Automated banning did not trigger as expected
- Control tuning required troubleshooting beyond initial configuration

## Security Concepts Reinforced

- Detective vs preventive controls
- Defense in depth
- Security control validation
- Continuous monitoring and control tuning

- Tune and retest Fail2Ban filters
- Validate automated response thresholds
- Build dedicated response-control testing lab
